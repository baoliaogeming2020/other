一、搜索引擎

提到搜索引擎，功能最全的还是谷歌，虽然还有Bing / Yahoo / yandex / duckduckgo等搜索引擎，但索引量最大的还应该是谷歌。其实，几个经济发展较好的国家(如：韩、印、日、德、法)都有自己的搜索引擎门户网站，也是可以进行参考的，毕竟每一个引擎的后台收录和搜索算法都是不一样的，所以对相同内容进行搜索获取的结果是不一样的。
这里特别提到yandex，是全俄文搜索，因此与大部分英文为基础的引擎有很大的区别；这里特别提到的duckduckgo是非主流搜索引擎，每次都会获取特别的搜索结果，而且这是onion推荐的搜索引擎，号称没有政治倾向。
这里特别提一点，中文搜索引擎有很多，当搜索中文信息时确实有一定的优势，但一定要小心使用。要注意以下几点：
  • 使用时一定不要登陆。
  • 如果有可能最好在翻墙的环境中搜索。
  • 搜索后最好用其它引擎二次确认。
某些中共历史信息在中国国内网站上可能已经无法找到(删除)，但国外有大量备份此类信息的网站。一但搜索到此类网站，记得收藏。以备后续搜索。

1. 特殊搜索

几乎所有的搜索引擎都支持特殊搜索方式，一种是“高级搜索”，另一种是“标记搜索”(我自己想的名字，实在是记不得叫啥了)。
高级搜索：是可以对搜索关键词进行增加修饰，从而达到精确命中目标的目的。但这种方式也有一定的坏处，就是将一些你认为的弱相关性内容排除在外，让你的思路没法打开。以谷歌为例：

如果你自己设定的环境为中文，进入界面都是中文，每个用法后边都有相应的解释，没有特别需要说明的内容。
标记搜索：一种类似与高级搜索的方法，但每次使用时不用进入高级搜索模式。这种方法学了一个搜索引擎，其它搜索引擎基本类似，因为这东西起源于谷歌，所以最好是学习谷歌的。
例如：“杨爛 site:baidu.com” 这是搜索在百度网站上所有与“黑森林”相关内容。
此类标记会根据引擎的不同会有一点区别，所以我不细讲了，只列出几个常用的标记：site / intitle / filetype / link / inurl
给大家找了一篇此类语法的文章，自己看吧。

2. 图片搜索

图片搜索是一个很少被用到的功能，但真的很强大。可以实现照片人脸识别功能，因此，对人的搜索有很大的帮助。使用也非常简单，只要把图片上传后，点搜索就可以。

二、书籍搜索

1. 搜索引擎

在搜索引擎中搜索，一般的关键词是："书名 filetype:pdf"，其中"pdf"也可以用其它格式替换，如：epub / mobi / azw3等。

2. 关键网站

我用的网站z-lib。无法给出链接，因为，此网站因版权问题经常更换域名，所以最好的办法是谷歌搜索"z-lib"。
此网站有近八百万书籍，而且增速极快，如：黑森林的《一问一世界》就是从这里找到的。

三、关键技术

magnet：磁力链接，一种可以实在P2P下载的技术，能够搜索到网上少见的资源。但好多链接因资源太少无法下载。但此类链接可以用迅雷转成种子(torrent)实现下载提速。
如：magnet:?xt=urn:btih:0c2d35c2f61e18a6370eadf471781e9fed5fdf3b
tracker：是一种P2P下载服务器，好多种子或magnet自带此类服务器，但此类服务器可能随时下线，因此，要更新，此类网站有很多，我只提供一个我常用的。newtrackon
BiglyBT：较好用的P2P下载软件，支持P2P的同时还支持I2P和tor下载。
