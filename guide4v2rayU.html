一、软件下载：

v2rayU是开源软件，主要发布网站在github.com，而github是中共大部分程序员学习编程的主要网站，因此，长期以来中共都不太敢封锁此网站。如要下载v2rayU请直接访问release，选择最新版本下载。因MacOS对非appstore程序安装有一定限制，所以，下载过程中一定要认真阅读此版本的提示信息，这些提示都是对安装有一定帮助的。

二、软件安装：

V2rayU已经是稳定版软件，但也会因某些原因出现小问题，如果最新版本在本机安装出现问题，请选择较早版本安装。
1. 双击下载的dmg文件，在安装界面中将V2rayU拖到Applications文件夹：

2. 打开启动台，找到V2rayU，点击启动。首次打开会有安全提示，点击“好”，如果软件无法打开，提示“V2rayU已损坏”，请打开终端输入如下命令修复：
sudo spctl --master-disable
sudo xattr -r -d com.apple.quarantine /Applications/V2rayU.app
 3. 然后打开“系统偏好设置” -> “安全性与隐私” ，点击下方的“仍要打开”，在新出来的授权弹框中输入登录账户密码，会再出现确认提示框，点击“打开”，V2rayU的图标出现在状态栏中。
(MacOS的状态栏就是整个屏幕最上边横条靠右的部分，其中第一个图标就是v2rayU。)

三、界面简介：

V2rayU这种翻墙软件在打开后没有主界面，只是一个常驻状态栏的下拉菜单，当点击某菜单项时会弹出窗口。

整个界面前两栏的作用不是很大：
"v2ray-core:on"表明软件的代理为开启状态。
"Turn v2ray-core:on/off"为软件开启与关闭菜单项。

1. 代理模式：

pac 模式：此模式下所有网络链接根据pac文件内的设定选择走代理还是直接连接，pac文件内写入大量被封锁网站，在此文件内的网站都通过代理连接。这就是为什么好多战友翻墙后可以访问油管却不能访问GTV，因为gtv.org没有写入pac文件内。
代理模式：此模式下所有网络链接都走代理。此模式下可能会无法访问某些国内网站，但基本可以访问GTV。
手动模式：不用管此模式，麻烦，少用。所以不解释了。

2. 服务器设置：

服务器列表：这里列出你翻墙的代理服务器节点。如果你刚安装应该是空的，因为你还没有加节点。
服务器设置：弹出一个窗口(如下图)，这里是手动添加节点的地方，刚安装，服务器列表单选框应该是空的。

导入模式：通过"url"或"file"一次性导入一个或者多个节点，如果选择url，一般在后边的框内填写"vmess://..."内容然后点击"导入"；如果选择file，再点击"选择文件"，可以选择每行一个vmess的文本文档，可以一次导入多个节点。
手动模式：用模式大部分是自建节点的战友，因为自建服务器节点后会生成各项信息，需要手动逐项填写。
如果没有自建节点的战友想试试此功能，可以在终端输入以下命令解析出节点的各项信息：
python -m base64 "vmess节点文本，不包括'vmess://'"
订阅设置：有好多战友帮助找了大量的订阅链接，这些链接内会有大量的免费节点，就是通过此功能导入软件的。此功能可以填写多个订阅链接。

在地址栏写入链接，备注随便写，点击"添加"按钮，会在下边增加一项，然后点击"更新"，会切换到更新界面。根据提标可以发现是否更新成功。
注意：
大部分订阅链接需要已经翻墙成功才能更新，因为墙内已经把这些订阅链接封锁。所以你一定要先有一个有效的节点手动添加到软件才能更新订阅。
要区分"服务器设置"与"订阅设置"，当你拿到以"https://"开头的链接文本时，用订阅设置，如果拿到"vmess://"开头的链接文本时，用服务器设置。
pac 设置：小白就别动了，前边讲过了。
ping speed：测试服务器的连通性，点击后，"服务器列表"下一级菜单每个服务器前的"-1 ms"会变成连通时间。其实这种测试不是很准确，不要认为时间短的就是连通好的服务器，因为ping通不代表tcp也是连通的。所以大家要多试几个节点。

四、其它操作：

从剪贴板导入：这是一个快速添加节点的方式。首先从文本文档复制"vmess://..."文本(可一次复制多个，但要保证每个一行，不包涵其它内容)，然后点击此菜单。如果失败会提示"import server fail:no found ss://,ssr:// or vmess://"，如果成功会提示"import server success:节点名"。
扫描屏幕上的二维码：打开二维码图片，点击此项菜单。不过有时会扫码失败。
分享二维码：此功能是通过电脑向手机分享的最佳方式，因为可以通过电脑快速测试出节点的速度，然后手机通过扫码添加此节点。也可以把"vmess://"复制后向其它战友分享。大部分自建节点的战友是通过手动添加的节点，向战友分享时，如果分享逐项信息比较麻烦，通过此方式分享非常方便。

五、其它问题：

1. 偏好设置 -> General -> 开机自动启动 最好选择，这样可以保证开机启动代理。
2. 偏好设置 -> General -> 启动或唤醒时自动从订阅更新 最好选择，这样可保持节点最新，避免节点失效。
3. 偏好设置 -> Advance 这里列出了代理地址及端口，当某些软件想通过代理连网时，可以按此标签内容填写。
4. 偏好设置 -> Pac 这里有个链接，是下载网上的pac文件，你可以自己修改链接。
5. 如何判断订阅链接的安全性：用浏览器打开链接，看到的文本以"dl"开头就是"vmess"链接，安全性不错，如果以"c3"开头就是"ss"或"ssr"链接，安全性不好。
