战友防止钓鱼攻击手册

一、网址与邮箱：

链接格式：

大概讲一下，后边要用，不细讲。链接的专用术语叫URL。
URL的一般语法格式为：(带方括号[]的为可选项)
protocol://hostname[:port] / path / [;parameters][?query]#fragment
例如： https://www.google.com:80/aa/bb?cc=dd;ee=ff#gg
战友常用到的有 http / https / ftp / file / mailto，其它还有好多，但在钓鱼攻击中很少用，所以也不讲了。http是最古老的网络链接，很不安全，如果战友看到这种链接最好不要点。https是替换http的新协议格式，已经对传输内容加密，但也分不同版本，最安全的是1.3，也就是最安全的"tls1.3"协议，但在链接中看不出来，要通过抓包软件才能看出来。ftp钓鱼中少用，但下载文件时可加载病毒。file是链接到本地文件，很少在网络链接中见到，但在某些下载的网页文件中可见，不太能用在钓鱼中。mailto是邮箱链接格式，这种格式钓鱼都是在收件时，或者是发件人中。

钓鱼攻防：

网址链接钓鱼一般是伪造"hostname"和"query"中跳转。
伪造：方式有两种，一种是注册类似"hostname"但后缀完全相同的域名，如："goog1e.com"，将字母"l"换成数字"1"。另一种是注册完全相同的"hostname"，但后缀不同，如："google.i2p"，因为域名后缀有很多，所以这方面是最要小心的。这两种方式都会伪造一个跟谷歌完全一样的页面，然后在这个页面里让你登陆，他在后台记录你的密码。
跳转：实现原理是因为某些网站可以上传文件，可以在此类文件中实现跳转，这种一般会在一个链接里套一个链接，如："https://www.google.com/redirect?url=http://ccc.ddd.com/eee"。如果你发现链接里套另一个链接，而且两个链接都不熟悉，就要小心了。这种一般是在安全链接里套一个不安全链接。有时后边这个假链接可能会通过一种格式转换，把":"转为"%3A"把"/"转为"%2F"，也就是当你看到链接里又出现"http%3A%2F%2F"时要小心了，可能是跳转。

其实还有其它伪造网站的方式，只是实现起来挺麻烦，战友判断也难，只提一下，如：是html中的frame嵌套，中间人攻击跳转。这些实现都需要有一定的条件，不太适用攻击我们战友。kali中有专门钓鱼工具，可快速伪造。原理跟我上边讲的差不多。

战友最好的防止钓鱼的方式就是看浏览器的状态栏。把鼠标放到链接上，不点击，这时浏览器就会给出链接的全文本，通过对文本的仔细研判就可以看出是不是钓鱼。有好多跳转链接特别长，在状态栏里看不全，链接里的嵌套链接可能会以省略号出现，这样的链接最好方法是，链接上点击鼠标右键，复制链接，将链接粘贴到文本中，慢慢查看安全性。某些无法分辨的字符可以通过改变字体看字符区别，如："l I 1"在不同字体中会有不同的区分。

邮箱链接钓鱼也是通过伪造，方式与链接伪造方式差不多，如：官方客服邮箱"support@aa.com"可以伪造成"support@aa.io"或者"service@aa.com"，第一种是通过伪造邮箱服务器，第二种是内部人士伪造注册邮箱。此类防钓鱼最好是到官网查询和战友的细心。

二、文件与后缀：

正常后缀：

一般网页链接都是.html / .htm / .php / .aspx / .jsp / .xml / .css 此类后缀通过浏览器下载到电脑时，都是纯文本，一般都安全，就算里边有危害性脚本，浏览器也会限制脚本访问权限。而后缀为 .exe / .bash / .jar / .dmg / .sh 都是可运行程序，下载后点击时要小心。而后缀为 .zip / .rar / .gz / .bz / .tar / .7z 一般都是压缩文件，此类文件解压后可能会存在隐藏文件，这样的隐藏文件可能存在危害。所以，下载后是压缩包一定要小心解压。图片后缀一般是 .jpg / .png / .bmp / .gif / .ico，此类后缀很难加载有害代码了。多年前曾经出现过图片预览漏洞，可以加病毒，但现在此类漏洞已经很难实现攻击，但也无法保证存在新的0day漏洞。

修改后缀：

如何判断改后缀文件：用UE之类的文本编辑器打开文件，会在第一行看到文件指纹，如：jpg图片会在第一行前几个字符看到"JPG"或"JPEG"或"JFIF"字符，如果你打开一个图片，看到的不是这样，就要小心了，至少不要双击。无法识别文件指纹的文件只有纯文本。大部分文件指纹与扩展名相同，只有长时间存在的文件格式，因为标准变动会出现不同指纹。

办公文档：

网上传播的word / pdf还是比较危险的文档类型，而且word的危险性比pdf危险性高些，因为，word宏是可以实现对硬盘的操作。pdf因阅读软件的种类多，不同软件出现漏洞的可能性也不一样，所以也是比较危险，同时，pdf文档核心模块可能类似，所以出现相同漏洞的可能性也不小。

三、节点与加密：

网上发布的免费节点与免费订阅，我一直跟战友说尽量少用，翻墙出来就尽量自建节点，这方面在好多手册里说了，不再啰嗦了。这种很可能钓鱼。此类钓鱼是通过降级攻击实现的，就是通过节点服务器将https降级到http，这也是为什么当你的节点不是tls加密时，此类降级攻击就可以实现，如果节点有tls加密，降级攻击就较难实现。所以一直劝战友用免费节点时，没有"tls"加密的尽量别用。

四、钓鱼加病毒：

大部分钓鱼攻击是收集战友的密码，上边讲的基本都是，但也有钓鱼过程中加载病毒的，这种攻击一但实现，战友的电脑就可能成为肉机，也可能被记录键盘，其实黑客手法极多，通过名称大家就能够知道会受到什么样的伤害了。一但受到此类攻击，只能重装电脑了，因为，现在能够感染硬件级别的病毒很少。就算是苹果电脑也最好重装。因为，能被病毒攻击，说明此病毒不被杀毒软件控制。想把病毒完全删除是一件很麻烦的事，特别是win系统。

五、社交媒体：

这是最危险的钓鱼攻击，大部分战友没有免疫力，而且一但中招，喝茶的可能性极大，喝茶后会被长期监控。
如：近期有伪类伪造"文斌"战友discord账号，向战友收集个人信息，如：真战友"Wood（文斌）"因为网名里有全角括号，很容易用空格加半角括号来伪造，战友要小心。所以，爆料革命管理员起网名时最好回避全角标点符号。战友在向管理员发个人信息时，最好细看管理员账号的字符大小写，标点的全半角，管理员账号的颜色，账号后的数字，如：discord会给每个账号后加"#098"这样的后缀。刚进入某群的战友最好观察一段时间再与群内交流，而且不要向任何账号发送个人信息。

五、其它方面：

少见的攻击方式：最好不要在翻墙时登陆国内账号，这样，会造成同一个IP翻墙账号被记录，但此类钓鱼不会面向非敏感战友，因为要监控的数量太大。此类攻击对那些自己建节点的用户危害更大。
