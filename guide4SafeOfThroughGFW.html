安全翻墙手册

手册概述：

如果你没被中共盯上，就不是敏感人士，用这个翻墙手册基本可以保证你的安全，就算在推特、油管等网站发些信息也可以；但如果中共盯上你，以下办法不适用，你只能肉身翻墙了。
翻墙最重要的是要把墙内墙外的各类信息严格分开，防止被中共收买的网站出卖你的信息。如果不能通过买国外手机卡将信息分开，也要通过邮箱跳转来保证墙内墙外信息只有一条通道。
其实在国内所有网络服务提供商都可以通过服务器发现你在翻墙，只是中共好多经济活动需要翻墙来完成，同时，国内好多科研机构也需要翻墙，翻墙人太多监控不过来，所以，就算你翻墙也没人管你，中共的原则是，没抓到你传播反共信息就没空理你。这里重要的是"没抓到"，外边那么多反共的，跟中国人无法对应中共是没法监控的。

硬件环境：

手机：翻墙尽量用苹果手机，二手手机就够用了，也不贵，最好是非大陆版手机。美区苹果ID是一定要有一个的，这种ID是可以自己申请的，只要把语言和地区改成美国再来一个美国IP就能申请了，填写地址时到谷歌地图上随便找一个美国地址就行。如果还没有翻墙设备，就先来一个台湾或香港账号，然后翻墙，然后再注册美国账号。翻墙手机不要安装匪区APP，不要用匪区产手机，因为已经有大量证据证明匪区APP在后台监控手机，匪区手机硬件留有后门。手机卡，这方面只能告诉敏感人士一定不要用，其它非敏感人士可以用，因为至少你还需要流量。
win电脑：不要安装一键Ghost操作系统，因为此类操作系统都安装大量后门，用此类操作系统你已经没有安全可言了，就算你认为自己找到了纯净版，里边也预装了360，这个软件是无法卸载干净的。如果你安装的是微软原版IOS包，就别安装杀毒软件和防火墙了，自带的防火墙已经够用了。
Mac电脑：国产软件不要给控制权限。其它也没啥交代的。
公共WIFI、免费WIFI：虽然所有的网络都知道你在翻墙，但免费WIFI是可以钓鱼的，同时可以通过中间人攻击、https降级攻击修改你的翻墙链接或者将你的加密链接转为非加密链接，这样你的安全性就没法保证了。

翻墙软件：

翻墙代理：VPN一定不要用，因为钓鱼的太多，好多都是中共在销售的VPN，如果你实在没有技术基础，只能给你个推荐expressVPN，这个还算安全，但VPN协议先天弱加密，没有混淆协议，现在的GFW已经能够检测此类链接。经过战友长期测试，现阶段安全成熟的翻墙软件是v2ray，也就是常见的vmess节点。此类节点也要选择性的用，最安全的是自己购买VPS再用一键安装脚本安装，安装后要着重看一下生成的配置文件文本中是否设置了TLS加密，没有此项安全性很难保证。网络上有大量的免费节点可用，但这些节点最好只当成翻出墙外的保障，一但翻墙成功最好自己建节点。最近出现vless节点，此类节点只是在vmess的基础上强制TLS，所以安全性更高。一种翻墙方式无法保证长期翻墙，所以最好用trojan / wireguard做为备份。主要用v2ray翻墙是因为它有混淆协议，对那些经常发帖和聊天的战友还是有一定的安全保证的。
操作系统：为防止黑客通过某些链接挂病毒的方式攻击战友，提醒大家要保持操作系统及软件的更新，因为，能够发现0day漏洞的黑帽黑客很少，大部分漏洞的发布规则是先发补丁再通过安全类网站发布漏洞攻击源码，一但源码发布，而你还没更新补丁，你被攻击的可能性就很大了。
浏览器：浏览器是翻墙最常用的软件，chrome / firefox / brave可以用，opera / 360 / 腾讯 / UC 等国产千万不要用。因为浏览器可以收集你的访问记录，而且可以收集你的登陆信息。如果战友不想死的太快，尽早放弃所有国内或跟国内公司相关的浏览器。
杀毒软件、防火墙：中国人常用的360，创史人就是流氓软件专家，后来被中共收编创立了360，这样的公司你是否能相信，你安装360其实就是主动给自己安装了监控，360是会在电脑空闲时扫描电脑硬盘的。其它一些国产杀毒软件也一样，战友尽早放弃。
虚拟机：好多战友会安装虚拟机来翻墙，可能是因为宿主操作系统不太会用，如：MacOS，也可能是为了安全考虑，但是，安装虚拟机后等于在一台电脑上组了一个小型局域网，当你的虚拟机不安全时，宿主机也就没有安全可言了，因为，局域网一但有一台机器被攻破后，其它机器也就没有安全可言了，中间人攻击是一种难度较低的攻击方式，还有好多局域网欺骗攻击。不幸的是，好多战友不会维护虚拟机的安全性。
模拟器：正写教程时，有战友问我menuplay可以翻墙，是否安全，查了一下，发现这个有中文网站，而且做了ICP备案，说明，menuplay公司为了推广中国业务搭建了翻墙服务器，但此类翻墙服务器的主要目的是保障游戏业务流畅，不是为了翻墙安全，所以，这种翻墙方式，只能告诉战友做为翻墙的备份。追问我，如果在模拟器里安装v2ray，这有点多余了，电脑里玩安卓就够费劲的。而且模拟器的安卓操作系统升级慢，可能会有大量漏洞。
APP：其实所有的APP都变相的收集用户信息，只不过是此类公司如何处理这类信息，中共国有法律，任何公司都必须将信息公布给国家，这样，你的信息就没有保证了。翻墙出来，先找找哪些APP不安全，国外已经禁止了好多中共国APP，并给出了禁止理由，搜索一下你就会主动卸载这些APP了，特别是在翻墙的设备上。
输入法：尽量不要用国产，如果一定要用也要关闭网络同步。国外的五笔输入法少。鼠须管还是比较安全的，因为是开源，但设置麻烦。所以，如果一定要用国产输入法，必须关闭同步功能。

账号邮箱：

邮箱三跳：如果你没有非大陆手机卡，也就是你所有的邮箱及账号都跟手机有相关性，一但某个账号被出卖，你的所有信息都暴漏了。你可以买那种不要身份证的手机号注册一套新的邮箱及账号，但这种是能够定位到你的位置的，其实外国手机卡走的也是国内塔台。所以，最安全的方法是，用手机到谷歌注册邮箱，再到protonmail注册邮箱，用谷歌邮箱验证，然后再找一个你喜欢的邮件服务器注册新的邮箱，用protonmail验证，所有后边的邮箱最主要的是不用手机验证。这样你的最后一个邮箱就可以去任何网站注册了，而中间跳转的邮箱定期登陆，保证不被封号就行。
账号分开：各类社交软件账号不要与国内账号有任何关系。密码不要相同，要用随机密码生成网站生成后再修改几位。随机密码网站有很多，随便找一个就行。FireFox在每次注册时也会生成密码，但不管是网站还是firefox生成的密码也不是完全可信的，因为有一种钓鱼叫彩虹表，所以这些生成的密码你随便改一俩位就行了。这种生成的密码都十分复杂，千万要记住。
TG / Whatsapp / Skype / Discord / Youtube / Facebook：有几款软件要手机号才能注册，其实大陆手机号就可以用，只是在加好友时只能你加他不能他加你，也就是不要把你的手机号给对方。
GTV：好多战友用QQ邮箱注册，我只问你一句，未来七叔说给某些利益时，腾讯不存在了，你的邮箱也没了，你拿啥证明这个账号是你的。

传播爆料：

加密压缩：最简单的是加密压缩传给国内战友，给个简单密码就行，然后用另一种方式发密码，如：微信发压缩文件，邮件发密码，因为网警一般都是通过文件散列值来判断的。这只是一般，如果你被监控了，所有文件都是要人工监控的。
百度网盘：如何在百度网盘传播，我做过教程，不再讲了，其实挺危险的，还是少用。
下载视频：YouTube-dl / m3u8 插件 / YouTube 下载网站，谷歌搜索一下，就有教程。
远程打印：国内远程打印千万不要用。
转发图片：旋转一定角度或在图片上加些点线，这样图片的散列值会变化，跟加密类似。

安全意识：

大部分黑客获取信息是通过钓鱼，不是真正用技术把你黑了。黑客能够用的手段，上边基本都提到了，所以你的安全意识比技术更重要。而大家常听说的DDos攻击，这种是没法防的，此类攻击有点象黑社会找一群人堵你家门，真的没法防。但这类攻击除了你上不了网，也没啥其它。但如果你在国内，说明你的IP已经暴漏了。他不知道你家门，怎么堵门呀。

断网保障：

Tor / I2P / TG：tor是老牌安全技术，但，现阶段只有联通网络速度可以接受，而且国内很难拿到初始节点。I2P，速度较慢，但连接有保障，安全性高与tor，而且那些免费节点订阅都可以通过I2P访问，做为翻墙最后保障很不错。TG，它有一种专有代理MTProto，此类代理数量极大，配置简单，不用翻墙，而且TG里有大量发vmess节点的群，所以，当成翻墙保障也不错。
Github：这是中共国内程序员必须要上的网站，中共不敢完全封锁，所以一定要关注此网站，这里经常会更新翻墙手段。现阶段安全的翻墙软件也都可以在此下载。
