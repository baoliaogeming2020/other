爆料革命网络宣传单制作手册

为什么要这样制做宣传单：网络宣传单要求是富文本(有图有文)，所以可选方案有：word / pdf / picture / markdown / html。其中 word / pdf 都有专有病毒，无法保证安全性，所以必须放弃。picture 已经很难加载病毒(历史上曾出现过图片病毒，但现在已经几乎没有了)，但图片中所有文字修改难，不可复制，传播过程中很容易被网监识别散列值，所以，很难在墙内传播，战友经常在图片上画点线传播，这样会破坏图片中的有效元素，所以不完全适合爆料传播。markdown 是一种类html语言，编写比html简单，但，现阶段还未实现全操作系统支持，也就是未流行，同时，也无法在同一文件内加载图片，所以，不完全适合。
而 html 做为成熟技术，熟悉此技术的人力资源足够，已经实现全操作系统支持，图文在同一文档加载，修改容易，本身是纯文本无法加载病毒，所有文本可复制，传播中对html标签较小的修改不会影响内容展现。因此是传播爆料的最好选择方式。同时，放到任何网站时，都可以与网站融为一体，大部分网络教程也很容易转成这种格式。这种方式做出来的宣传单随时可以生成图片传播。

注意事项：

1. 不加脚本：现在脚本已经很难实现对操作系统的攻击，但链接还是可以通过跳转钓鱼的。所以在宣传单内不加任何脚本类元素。
2. 写入图片：所有图片以 base64 文本形式写入html，这样整个宣传单会以单一文件形式存在，传播时不会出现图片无法加载。
3. 少加链接：链接跳转有钓鱼嫌疑，因此尽量要少加，如果一定要加，必须是可信链接。我会监督群里此类宣传单中的链接。
4. 写入样式：样式表不用link，直接写入html文件内，保证美观，传播中又不丢失文件。要尽量保证样式表内容简洁、有效。

实现方式：

1. 编写规则：html的编写就不多讲了，只提一下，各农场可以根据自己的特点和实力做一套模板，并对所有以此模板发出的文档进行编号管理，防止匪类借此宣传。所有文档编码一定是utf-8编码，这样适用全世界所有语言操作系统浏览，不会出现乱码，也就是『<meta charset="utf-8">』。
2. 图片加载：用命令"python -m base64 aa.png"将图片生成base64文本，此方法在 mac / win / linux 上都适用，只不过 win 需要安装python。然后，在html内写入『<img src="data:image/png;base64,...">』(省略号代表刚生成的base64文本)，我个人倾向gif图，因为压缩较好。
