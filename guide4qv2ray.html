一、软件下载：

Qv2ray是开源软件，主要发布网站在github.com，而github是中共大部分程序员学习编程的主要网站，因此，长期以来中共都不太敢封锁此网站。如要下载Qv2ray请直接访问release，选择最新版本下载。同时，此软件下载版没有将v2ray-core包涵在下载包内，要到vcore下载，请根据自己的操作系统选择下载。

二、软件安装：

MacOS

1. 双击下载的 dmg 文件，在安装界面中将 Qv2ray 拖到 Applications 文件夹。
2. 在您首次启动 Qv2ray 时，您可能收到一条警告称 Qv2ray 是从网络下载的文件。您是在 Qv2ray 官方网站下载的，所以只需点击"打开"。或者，打开“系统偏好设置” -> “安全性与隐私” ，点击下方的“仍要打开”，在弹出来的授权弹框中输入登录账户密码，会再出现确认提示框，点击“打开”。
3. 然后 Qv2ray 的图标出现在状态栏中，其中第一个图标就是 Qv2ray。

WinOS

1. 双击下载的 exe 文件，一路"下一步"就可以安装成功了。
2. 进入 WinOS 开始菜单(或者在桌面)，找到 Qv2ray 的 logo，点击启动。
3. 然后 Qv2ray 的图标出现在系统托盘中，其中第一个图标就是 Qv2ray。

加载核心

所有的v2ray界面版软件，都是以v2ray-core为基础开发，只是将v2ray-core以界面形式加载，而v2ray-core经常更新，所以好多界面版作者未将v2ray-core放入发行包内。需要您自己下载并加载。以下的设置是官方建议设置，你也可以直接随意解压到一个文件夹然后选择此文件夹就可以。也可以按照"内核设置"内的两个文件夹路径自建文件夹放置。
1. 将 v2ray-core 下载的压缩包解压到 vcore 文件夹。
2. 首选项 -> 内核设置 -> v2ray核心可执行文件路径 -> "选择"按钮 -> 弹出窗口选择vcore文件夹内的v2ray(.exe) -> 打开。
3. 首选项 -> 内核设置 -> v2ray资源目录 -> "选择"按钮 -> 弹出窗口选择vcore文件夹 -> 打开。
4. 首选项 -> "OK"

三、简单翻墙：

1. 从其它软件复制"vmess://..."文本，可一次复制多个，如：我发的"vmessAll4KillCCP.html"文档，如果文档内包涵其它信息，无法一次性复制多个，可以将节点一个个复制到其它文本文档内，再一次性复制。因为不是所有节点都能穿透你的网络，最好多加几个节点。
2. 主菜单 -> 显示。显示主窗口。
3. 主窗口 -> 新建。显示"导入文件"窗口。
4. 导入文件 -> 链接 -> 分享链接文本框内粘贴刚才复制的 vmess 节点文本。
5. 导入文件 -> 导入。关闭"导入文件"窗口。
6. 主窗口 -> 双击"Default Group"组。展开本组所有节点。
7. 双点某个节点。
8. 主菜单 -> 系统代理 -> 启用系统代理。
9. 如果无法翻墙，或者速度慢，重复步骤1 - 7，选择其它节点。

四、界面简介：

Qv2ray 这种翻墙软件在开启会打开主界面，并常驻状态栏(系统托盘)图标，但在正常翻墙时主界面没啥用，直接关闭就可以，不过首次安装需要设置，图标点击会有下拉菜单。

主菜单

显示/隐藏：显示隐藏主窗口。
首选项：显示设置窗口。
系统代理：设置系统代理，也就是大部分软件不用设置直接可以翻墙。因为好多软件，如浏览器，已经设置为自动走系统代理。
最近连接：连接过的节点服务器。
连接：连接节点服务器。
断开连接：断开节点服务器。
重新连接：重连。
退出：退出软件。

主窗口


每次启动都会打开主窗口。首次安全后，分组列表内已经存在"Default Group"。但分组内没有节点，需要添加。
添加节点有四种方式：1. 通过填写 vmess 链接添加；2. 通过二维码添加；3. 通过订阅链接添加；4. 手动添加(不讲)。单独节点与订阅链接不能放到同一个分组内。

添加节点

1. 主窗口 -> 新建。
2. 导入组中 -> 下拉选择"Default Group"。

vmess方式

3. 从其它软件复制"vmess://..."，可以一次复制多个，但要保证每行一个。
4. 链接 -> 分享链接 -> 将复制内容粘贴到文本框内
5. 导入。

二维码方式

3.1. 选择"隐藏 Qv2ray"。
3.2. 点击"开始"，然后鼠标左键按下选择屏幕上的二维码区域，再点"截屏"。
3.3. 或者，点击"选择"，打开二维码图片文件。
4. 导入。

订阅方式

1. 主窗口 -> 分组。弹出"组编辑器"窗口。
2. 点击"组列表下"的"新建"按钮。
3. 分组名称文本框内随便输入一个名字，也可以不改。
4. 点击"订阅设置"，选择"此分组是一个订阅"。
5. 订阅地址文本框内输入订阅链接。
6. 点击"更新订阅"按钮，某些订阅链接必须要先翻墙才能更新，所以当第一次用时，不一定能更新成功。
7. 点击"OK"按钮。

五、常见问题：

1. 如何判断订阅链接的安全性：用浏览器打开链接，看到的文本以"dm"开头就是"vmess"链接，安全性不错，如果以"c3"开头就是"ss"或"ssr"链接，安全性不好。
2. 自建节点无法连接最可能问题：最可能的是服务器与本机时间不协同，因v2ray要求时间差在2min中内，所以，如果你本机时间是网络同步，就要改服务器节点的时间。也有部分战友本机时间是手动设置，甚至年月都与当前时间不匹配。
3. 什么是翻墙的最后一道保障：答案是telegram或者i2p。因为telegram有一种mtproto代理，这种免费的独特代理数量及其庞大，好多telegram群为了推广都会不断推出此类代理，同时有大量发布vmess节点的群。i2p，类似tor，但不用获取节点。也有些人说最后一道保障是其它，我解释一下，比如：tor，国内只有联通速度还可以，但难以获取节点。

六、节点格式：

v2ray翻墙节点有6种格式，这些格式一方面方便了节点的传播和添加，但也让大部分战友迷惑。获取错误格式通常会在添加节点时提示错误。
1. 通用链接格式："vmess://..."，此种格式全部内容都是base64加密，无明文，也就是看不懂的乱码。这种格式适用大部分v2ray软件。
2. kitsunebi格式："vmess://...?network=tcp&aid=0&tls=0&allowInsecure=1&mux=0&..."，此格式前半部分是base64加密，后边是明文参数，也就是"?&a=b"。此类格式只适用kitsunebi。
3. BifrostV格式："bfv://45.63.34.187:50750/shadowsocks/1?rtype=lanchina&dns=8.8.8.8&method=..."，此格式全是明文参数，此类格式只适用BifrostV。
4. v2ray订阅链接：订阅链接内容是对多个vmess再次base64加密的文本以http或https发布，一般都是"https://..."。此种链接里的内容也可能会出现以上三种vmess的格式，大部分都是第一种，也有把SSR加入其中的。
5. clash订阅链接：clash专用的订阅链接内容不详细说了。这种格式太大了。此格式只适用clash。
6. 二维码：1 - 3都可以生成二维码，但不同格式生成的码只适用于原格式软件扫码。
