战友如何参与挖掘中共老底

全球互联网包括明网和暗网，其中明网最大的搜索来自于搜索引擎对明网的收录及索引，而暗网没有成体系的收录及索引，但可以通过对暗网的了解找到部分索引，将多个部分索引关联后形成一个尽量完整的索引，这种能力的提升需要对互联网及网络软件的不断熟悉来进行能力提升。
本教程只讲解通过互联网如何快速搜索来完成任务。以近期的“黑森林事件”为例进行讲解。

一、搜索引擎

提到搜索引擎，功能最全的还是谷歌，虽然还有Bing / Yahoo / yandex / duckduckgo等搜索引擎，但索引量最大的还应该是谷歌。其实，几个经济发展较好的国家(如：韩、印、日、德、法)都有自己的搜索引擎门户网站，也是可以进行参考的，毕竟每一个引擎的后台收录和搜索算法都是不一样的，所以对相同内容进行搜索获取的结果是不一样的。
这里特别提到yandex，是全俄文搜索，因此与大部分英文为基础的引擎有很大的区别；这里特别提到的duckduckgo是非主流搜索引擎，每次都会获取特别的搜索结果，而且这是onion(暗网)推荐的搜索引擎，号称没有政治倾向。
这里特别提一点，中文搜索引擎有很多，当搜索中文信息时确实有一定的优势，但一定要小心使用。要注意以下几点：
  • 使用时一定不要登陆。
  • 如果有可能最好在翻墙的环境中搜索。
  • 搜索后最好用其它引擎二次确认。
  • 搜索引擎其它产品使用时不要标注个人信息，如：百度地图不要标注个人家族、单位地址。
某些中共历史信息在中国国内网站上可能已经无法找到(删除)，但国外有大量备份此类信息的网站。一但搜索到此类网站，记得收藏。以备后续搜索。

1. 特殊搜索

几乎所有的搜索引擎都支持特殊搜索方式，一种是“高级搜索”，另一种是“标记搜索”(我自己想的名字，实在是记不得叫啥了)。
高级搜索：是可以对搜索关键词进行增加修饰，从而达到精确命中目标的目的。但这种方式也有一定的坏处，就是将一些你认为的弱相关性内容排除在外，让你的思路没法打开。以谷歌为例：

如果你自己设定的环境为中文，进入界面都是中文，每个用法后边都有相应的解释，没有特别需要说明的内容。
标记搜索：一种类似与高级搜索的方法，但每次使用时不用进入高级搜索模式。这种方法学了一个搜索引擎，其它搜索引擎基本类似，因为这东西起源于谷歌，所以最好是学习谷歌的。
例如：“杨爛 site:baidu.com” 这是搜索在百度网站上所有与“黑森林”相关内容。
此类标记会根据引擎的不同会有一点区别，所以我不细讲了，只列出几个常用的标记：site / intitle / filetype / link / inurl
给大家找了一篇此类语法的文章，自己看吧。

2. 图片搜索

图片搜索是一个很少被用到的功能，但真的很强大。可以实现照片人脸识别功能，因此，对人的搜索有很大的帮助。使用也非常简单，只要把图片上传后，点搜索就可以。

二、书籍搜索

1. 搜索引擎

在搜索引擎中搜索，一般的关键词是："书名 filetype:pdf"，其中"pdf"也可以用其它格式替换，如：epub / mobi / azw3等。

2. 关键网站

我用的网站z-lib。无法给出链接，因为，此网站因版权问题经常更换域名，所以最好的办法是谷歌搜索"z-lib"。

三、关键技术

magnet：磁力链接，一种可以实在P2P下载的技术，能够搜索到网上少见的资源。但好多链接因资源太少无法下载。但此类链接可以用迅雷转成种子(torrent)实现下载提速。
如：magnet:?xt=urn:btih:0c2d35c2f61e18a6370eadf471781e9fed5fdf3b
方法：打开迅雷，复制magnet链接，迅雷会自动提示下载(如果不提示，说明你没打开迅雷的"监视剪贴板"，用完记得关，让墙内软件监视剪贴板不是啥好事)，点击确认，会在迅雷里增加一个下载任务，右键此任务，打开此任务的文件夹，在此文件夹中显示隐藏文件(搜索引擎搜索方法，很好学)，可以找到.torrent文件，这就是种子，用其它软件下载。
tracker：是一种P2P下载服务器，好多种子或magnet自带此类服务器，但此类服务器可能随时下线，因此，要更新，此类网站有很多，我只提供一个我常用的。newtrackon
BiglyBT：较好用的P2P下载软件，支持P2P的同时还支持I2P和tor下载。上边说的tracker网站就是给这类软件用的，因为，好多种子里写的服务器不够，或者已经关闭，需要增加或更新。P2P软件有很多，根据自己的习惯使用。
Scarangel：这是一个比较强大的，ed2k网络搜索工具。因P2P技术特点造成的文件消失慢，所以，在P2P网络中的文件大部分都会留下印记。此类搜索在网上很少有教程，我曾经为了熟悉此类网络，玩了一年。大概跟战友说一下方法，以一个关键词切入，找到相关文件，文件在下载过程中会连接到用户，在用户点右键可以查看此用户所有正在下载的文件，在这些文件中查找相关性强的文件，这样逐步跟踪，会找到大量相关性文件。还要重点说一下，连接不用的服务器，搜索结果不同，不同时间的搜索结果也不同，所以，当以某个关键词切入时，要在不同时间进行多次搜索，或者，在连接不同服务器时，进行多次搜索。在此类网络中，有大量黑客放上去的攻击结果文档，如：某某宾馆的开房记录。

四、搜索引擎的搜索

搜索是一个比较庞大的体系，而且在不断更新，因此，当想提升自我搜索能力，就要先用搜索引擎来搜索"搜索引擎"、"十大搜索引擎"、"搜索与SEO"等内容。

五、关键词提炼：

关键词：

关键词的提炼是信息挖掘中挺重要的一个能力，但这种能力不是特别难练习，一般就是在搜索某个人时，要通过初期搜索提炼相关的 公司、地址、邮箱、电话等信息，对此类信息进行二次搜索，这样逐步扩展就可以全面收集此人的信息。而且，也要有一点社会工程学能力，比如：当你搜索的是一个未婚的年青女性时，就要想到她可能会在相亲网站留下一定的信息，而此类信息的切入点可能是她的英文名。因为，大家都不想把真实姓名留到网上。这里我只拿搜索个人举例，其它方面的搜索要自己练习。

翻译：

因同样信息可能会用不同语言报道，所以，当搜索某关键词时，需要搜索不同语言，这样就涉及到翻译，目前最好的还是谷歌翻译，但好多其它翻译网站可以当成辅助，毕竟谷歌有时也作弊。翻译过程中，要对翻译结果逐步校准，因为，此语言毕竟不是你的母语，你不是十分了解母语国家如何描述此事，所以，在搜索过程中要不断向母语国家的描述靠近。

六、类搜索引擎：

信息分类：

此类网站大部分是某类信息的收录集合，比如：企查查是企业信息收录集合。此类信息的搜索需要对互联网的了解，不过也可以通过搜索引擎来查找，但此类查找也需要不断锻炼关键词提炼。

搜索功能：

大部分网站都有搜索功能，但有此功能的网站大部分都是挂载数据库，造成搜索引擎的收录不全，所以，就要根据经验进入到相关网站找到搜索功能进行搜索。

七、黑客手段：

谷歌黑客：

黑客有专门一课叫"谷歌黑客"，这是通过谷歌的收录功能将好多网站的漏洞收录，通过某个关键词的方式可以找到某些未打补丁的网站，找到这些漏洞后就可以进入此类网站的后台拉取无法被搜索引擎收录的信息。
谷歌黑客不需要太强的技术，一本教程就能学会

数据库拉取：

这个挺难的，一般是通过 SQLinjection 完成的，这是需要较强的技术功底。而且要不断学习新的技术。

XSS：

也是一种黑客手段，通常用于社交媒体进行钓鱼，通过构造脚本等待目标人去点击来获取目标人的信息，甚至密码。

查看源码：

某些网站用脚本技术禁止了文字拷贝和图片下载，这时用浏览器查看源码就比较重要了，如谷歌的"开发者工具"，火狐也有相关功能，不记得叫啥了，国内好多chrome内核浏览器也有相关功能，但最好不要用，因为，中共会在源码里加监控。

八、其它暗网：

其实暗网不只 tor 一部分，还有 I2P / freenet 等等好几个，同时，还有人将暗网定义为搜索引擎收录以外的网络，所以，想玩搜索，特别是敏感信息、隐藏信息的搜索，对这几部分暗网的了解也挺重要的。这部分是需要时间磨的。
