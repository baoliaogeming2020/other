I2P：是一项混合授权的匿名网络项目。 I2P 网络是由 I2P 路由器以大蒜路由方式组成的表层网络，创建于其上的应用程序可以安全匿名的相互通信。它可以同时使用 UDP 及 TCP 协议，支持 UPnP 映射。其应用包括匿名上网、聊天、撰写博客和文件传输。
BiglyBT：是基于 Azureus 开源项目的功能齐全的 Bittorrent 客户端，支持 public i2p tor 网络 BT 分享与下载。安装"I2P Helper"插件，支持匿名 I2P 网络下载。这包括 DHT 对分散跟踪的支持。与"Message Sync"插件结合使用时，支持匿名聊天。

二、主要内容：

1. 软件安装：

1.1. 软件下载：

BiglyBT 是开源软件，主要发布网站在 github.com，而 github 是中共大部分程序员学习编程的主要网站，因此，长期以来中共都不太敢封锁此网站。下载请直接访问 release，也可以到 官网 下载。根据操作系统选择最新版本下载。本软件是由Java编写，在安装前需要安装 Java Runtime Environment(JRE)。BiglyBT 的 I2P Helper 插件已经将所有 i2p 的 Bittorrent 功能写入，不需要再安装 i2p。只需要下载并安装本插件。

1.2. 软件安装：

1. 先安装 JRE，安装方法网上太多，不多介绍，基本上是一路"下一步"就完成了，Mac 可能会出现输入管理员密码的窗口。
2. 再安装 BiglyBT，跟所有操作系统软件安装差不多，Mac 会有点差别。Mac 下软件大部分是拖拽式安装，而此软件是一步一步安装，也基本是一路"下一步"就完成了。
3. 最后是 I2P Helper插件，直接双击下载的插件文件，然后重启 BiglyBT，就会发现已经安装完成。如何判断安装成功：首选项 -> 插件 中出现 "I2P Helper" 项。

1.3. 界面介绍：

1.3.1. 主菜单：

BiglyBT -> Preferences：首选项菜单，对软件进行设置。win/linux将此菜单放置在"工具 -> 选项"。
文件 -> 打开 -> Torrent文件：打开种子或磁力链接，会弹出文件选择窗口。
文件 -> 共享 -> 文件：打开共享文件，弹出文件选择窗口。
查看：本菜单下子项会在侧边栏增加一项。也可能会弹出子窗口，或者改变主窗口显示。
工具：本菜单下子项会在侧边栏增加一项。"我的共享"子项为所有共享文件管理。

1.3.2. 基础设置：

切换语言：首次打开时主界面为英文，"首选项窗口 -> 界面 -> 语言 -> 中文(中国)"切换为中文。
设置共享："首选项窗口 -> 共享 -> 网络"选择"公网IP网络(不匿名)"和"I2P匿名网络"。

1.3.3. 主界面：

本教程以MacOS安装界面为例，WinOS 或 Linux 界面略有差别，但差别不大。

侧边栏
侧边栏：主窗口左侧，以二级列表形式展示软件主要功能。
侧边栏 -> 我的torrent -> 我的媒体库：显示所有正在下载/分享的进程。
侧边栏 -> 我的torrent -> 我的共享：只显示正在分享的文件。双击在"我的torrent"下级打开分享进程项。
进程列表
功能区：当点击侧边栏某项时，功能区会根据侧边栏切换。此处只讲解下载/分享进程功能区。
功能区 -> 进程列表：显示下载/分享进程列表，选择某个进程会在"操作标签"加载本进程的操作项。每个进程点击右键会弹出右键菜单。
功能区 -> 进程列表 -> 右键菜单 -> 控制 -> Tracker/Torrent选项 -> 添加Tracker地址：向种子增加Tracker，此类Tracker可通过newtrackon.com找到，以增加公网下载发现内容的可能性。
功能区 -> 进程列表 -> 右键菜单 -> 控制 -> Tracker/Torrent选项 -> 在Finder中显示：打开种子所在的文件夹，当分享时非常有用，软件会自动生成种子，可以打开种子所在文件夹。
功能区 -> 进程列表 -> 右键菜单 -> 控制 -> 强制开始：当下载进程数较多时，有部分进程进入"排队中"状态，可点击此菜单按钮强制下载；当分享进程数较多时，有部分进程进入"排队中"状态，可点击此菜单按钮强制分享。
功能区 -> 进程列表 -> 右键菜单 -> 控制 -> 高级 -> 超级做种：此模式下，优先给连接用户传输从未上传的快，此模式保证分享进程的完整性。
功能区 -> 进程列表 -> 右键菜单 -> 控制 -> Tracker/Torrent选项 -> 编辑Tracker地址 -> 添加列表：在文本框内填写"https://newtrackon.com/api/stable"，确定，保存。可以向本进程添加 Tracker，增加可用下载源。
操作标签
功能区 -> 操作标签：分为多个标签，每个标签对应不同的功能，可对当前选择进程进行相应功能的操作或显示。每个标签内的文本点右键都会弹出复制菜单，可复制内容文本到剪贴板。
功能区 -> 操作标签 -> 常规：显示进程基本信息。
功能区 -> 操作标签 -> 选项：设置进程上传/下载速度。
功能区 -> 操作标签 -> 隐私："策略级别"中设置本进程通过公网/I2P进行传输。"用户数:Public=4, I2P=1, 传入连接=0, 传出=0/5"显示连接种类用户数量，当"传入连接"不等于0时，基本就是找到下载源，或有人在下载此文件。"I2P"中"可用性"后的按钮一定要禁用(灰色不可点)才表示你的"I2P helper"插件安装成功。"搜索DHT"按钮帮助搜索可用下载，如果搜索几次后，在其后的第二个文本框内无内容，说明无下载源。
功能区 -> 操作标签 -> 聊天：在此标签内根据本进程建立聊天群，同时会根据不同语言，不同tracker建立公共和匿名聊天群。在此处可互相沟通本进程内容及如何提速。
功能区 -> 操作标签 -> 区块映射：此标签表示本进程文件的每个分割块的信息，用颜色表示。在分享过程中，当大部分块上的数字为1或2时，表示正在分享此进程的用户数量较少，最好不要关闭分享，也不要删除进程。
功能区 -> 操作标签 -> 信息 -> 分享率：本项信息表明本进程分享内容完成情况，当值为"<1"时表明分享内容没有用户下载完整；"=1"表明至少有一名用户下载完整；">1"表明至少一名下载完整，有其它用户下载部分。因此，如果删除本进程，一定要保证此项值">1"。

2. 分享方法：

因电脑有限，无法进行多点分享测试，各主播最好建立分享小组进行BT分享，这样可以保证不同时区的战友在分享，多个战友在同一时间分享可以保证下载速度。
文件 -> 共享 -> 文件：打开文件选择窗口，选择要分享的文件并确定关闭窗口。"功能区 -> 进程列表"中选择此共享文件，保证"功能区 -> 操作标签 -> 隐私 -> 网络"中的"I2P匿名网络"为选择状态。如果你有独立IP，而且个人安全有保证，也可以把"公网IP网络(不匿名)"选中，如果没有独立IP，此项选中也没用。提示：公网选中会暴露你的IP。
1. 分享Bittoront种子："功能区 -> 进程列表 -> 右键菜单 -> 控制 -> Tracker/Torrent选项 -> 在Finder中显示"打开种子所在文件夹，将种子向战友传播。也可以将多个种子打包并增加注释文本文档再传播。
2. 分享Magnet链接："功能区 -> 进程列表 -> 右键菜单 -> 社会化 -> 复制磁力链接到剪贴板"发送给战友进行传播，也可以将多个magnet整理到一个文本文档中，加上注释进行传播。
3. 本软件聊天群管理："功能区 -> 操作标签 -> 聊天"显示本进程聊天群，对此进行管理，可加速分享。让墙内战友了解本进程主要内容。
4. 本软订阅内容管理："功能区 -> 进程列表 -> 订阅"管理本进程订阅，不断更新，让下载此进程的战友，可以随时关注下载列表更新。此功能需要其它网站链接辅助。

3. 下载方法：

你的下载，也是向其他战友传播，下载后最好有一段做种时间，保证全球战友的下载速度。请不要快速删除进程。
1. 文件 -> 打开 -> Torrent文件：打开窗口，如果拿到的是magnet，则在文本框内粘贴magnet链接；如果拿到的是种子，点击"添加文件"按钮选择种子文件。点击"确定"按钮。
2. 功能区 -> 操作标签 -> 隐私 -> 搜索DHT：多次点击此按钮来确认可用源，软件会自动搜索可用源，但通过此按钮可以快速确认可用源数量，一般十分钟内多次点击可发现源。

4. 传播方法：

传播时，一定要把教程一并传播出去。因为此软件在各大搜索引擎也很难找到详细教程。
1. 聊天软件传播：虽然是文本或种子小文件，如果用匪区聊天软件传播时最好还是进行一次加密。
2. 建立网站传播：其实不一定要个服务器那么麻烦，只要有一个能将种子上传，或者magnet文本提交的服务就可以。最好是匪区没有封锁的网站，如：github.com。
3. 墙内下载传播：墙内所有聊天软件对内容的判断基本是通过hash来判断屏蔽的，所以墙内战友在向墙内战友分享时，如果通过匪区聊天软件分享，最好进行简单加密压缩，或者用UE之类的软件对视频二进制进行简单修改。

三、其它方面：

与BiglyBT来自同一个开源项目的软件vuze，可以完成相同的功能。用法也基本一致。 BitComet(比特彗星)也可以支持i2p的BT分享，有兴趣的战友可以研究一下。不是原生。
如何用UE修改视频：视频点击右键用UE打开，可以看到视频十六进制，在视频中出现"00"最多处随意修改几个字节，千万不要添加字节，可能会改变文件整体结构。或者，在文件最后增加或修改几个字节。这样不影响原视频播放，但hash码已经变化，不会被匪区聊天软件识别，此方法也适用于向百度网盘上传。但这种传播多了，也可能被识别，所以，最好的办法还是简单的加密压缩。
可替换迅雷的下载软件：经过短期测试发现，此软件是完全可以替换迅雷的下载软件，大部分迅雷没有速度的进程，此软件会有少放速度，而此软件没有速度的，迅雷基本都没有速度。只是部分匪区流行的内容，迅雷比此软件速度快些。如果战友有经常从BT下载的需求，可试着转到此软件，通过长期使用，也可以逐步摸索传播爆料革命的快捷方式。
