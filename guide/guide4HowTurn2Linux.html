爆料革命战友的安全特别是墙内战友的安全是未来中国实现民主的火种，任何一位战友都不要妄言"不怕死"。现阶段还没有到墙内游行的时机，每一位战友的牺牲都是爆料革命的重大损失。一定要记住"中共不怕你牺牲，就怕你的牺牲引起墙内的振动"。
win操作系统对于战友的安全是个巨大的威胁，因此，战友要逐步远离微软，转向较安全的linux或mac。有好多技术小白战友常说"我没有技术基础，根本不懂电脑，怎么转呀？"。我会在这里教战友如何无缝转向。这种转向不但能够增加战友的安全性，还可以降低硬件投资，而不在乎金钱的战友能够转向高端的MAC电脑。通过这种方式可以让战友转变"win是应用最多最好用的操作系统"的错误思维。在未来的办公在获取更好的体验，能够快速升职。

如何逐步远离微软

为什么要远离微软：微软做为最早进入中国的公司，与中共国勾兑时间久远，盖茨也是一个极左。同时，微软的不开源与苹果不同，微软系统中有好多来自第三方的模块，而这些模块一但来自中共，则无法保证安全。微软与linux的开源没法比较安全性。
如果资金较敏感的战友在买电脑时一定会考虑操作系统，如果安装正版则要增加投资，安装盗版，则一般是"gho"版本，安全性很难保证(这一点我在gnews中谈过)。然而，安装linux则没有投资成本问题。
如果资金不敏感的战友在买电脑时如果购买MAC电脑，则能够达到一步到位的感觉，而且降低垃圾软件的打扰。因为，苹果电脑所有的软件都是根据自己的硬件进行优化，能够保证软件的运行稳定，同时，苹果基本从底层防止垃圾软件的泛滥。

没有微软真不行吗：如果你是一名非技术人员，电脑只是为了娱乐和办公，十年前说"没有微软直不行"，这种说法还真的有道理。因为，当时的linux以面向技术人员和服务器为主，此操作系统缺少很多基础办公和娱乐软件，用此操作系统的技术人员一般都是在玩命令行来完成办公。但经过长期的发展，linux已经将重心从专业向民用转变。这个转变的同时依然保持着操作系统的专业性，在原有专业的基础上发布了大量的民用基础软件。同时，有好多战友手里有比较老的电脑也是可以安装linux操作系统的。linux对硬件的要求比较低，因为linux没有windows那么臃肿，大部分linux内核都是用C直接写的。
十年前的MAC电脑也存在相同的问题，只不过MAC的定位是美工、出版等专业行业，苹果将各类硬件与软件的优化到极致，将PC无法完成或很难在个人电脑上完成的工作，用MAC完美的实现，比如：视频编辑、3D渲染等。但在乔布斯最后一次回归苹果公司时，将MAC的普及做到了极致。
随着linux和mac不断的从专业向民用转变，市面上大量主流软件公司转向同时发布linux和mac版本，最好的证明就是微软Office已经发布了Mac版本。

一、双操作系统：

安装双操作系统是无缝转向的最好方法，这样可以在不影响以往习惯和工作的情况下逐步向另一个操作系统转变。

虚拟机：

虚拟机就是在一个操作系统里用软件模拟另一个操作系统的硬件环境并安装，这样就可以同时运行两个操作系统，一个是习惯的，另一个是要转向的。
平时玩PC的可以用虚拟机模拟linux操作系统，而买了MAC电脑的可以用虚拟机模块windows，这样就可以慢慢的转向。通过虚拟机慢慢习惯另一个操作系统就可以将原习惯的操作系统做为辅助，如：在PC上安装linux，在虚拟机里安装windows，或者选择后边的双启动。
主流虚拟机软件VMware / VirtualBox / Hyper-V。

双启动：

双启动是指在打开电脑时显示操作系统选择，这样在一台电脑上就可以安装两个操作系统，开机时选择启动哪个操作系统。此方法对于非技术人员转换操作系统不是特别适合，但后期，习惯新操作系统一段时间后，对电脑的速度提升也一定好处。
MAC电脑：Boot Camp用于帮助用户在内置Intel处理器的苹果电脑上安装微软的Windows系列操作系统。
PC电脑：一般可以直接安装双系统，因为主板就支持双启动。还可以选择用U盘安装linux，在启动时选择用U盘启动linux，也可以直接启动windows。

操作系统选择：

Mac没得选，买回来就带操作系统。你要做的就是习惯，然后将软件也慢慢转到这里。Mac唯一可以选择的就是操作系统的版本，当你升级到某一个版本发现不太好用时，如何降级。这里不细讲，大家只要搜索"Mac U盘安装"就能找到大量教程。
Linux操作系统有很多，推荐新手可以学Ubuntu和Debian，因为这两个操作系统在操作上逐步向微软靠拢。网上教程也相对较多，发布的软件数量较大，在转向的过程中较少会遇到无法替换的问题。中文界面完整，又没有中国公司背景。

二、软件转换：

软件是大家最关心的，因为好多人认为较多的windows下常用的软件在Mac和linux下没有替代软件。在这里以我个人经验告诉大家，只有游戏是无法替代的，其它软件都有相应的替代品，还有就是中共国开发的软件有部分是没有其它操作系统版本的。

1. 办公软件：

最常用的就是Office，其实前边也说了，已经出了Mac版，而且功能基本一致。而在linux下有同类的办公软件很多，Apache OpenOffice / Feng Office / Siag Office / Calligra Suite / ONLYOFFICE / LibreOffice。这些基本上都可以完美的替换微软Office，而且都可以将微软Office文档打开或转存。界面和操作也基本类似。
Mac有自己的办公软件iWork，包括：文字处理和排版软件Pages，幻灯片制作展示软件Keynote和电子表格软件Numbers。据说比微软的好用，而且界面也更加美观，特别的幻灯片，实在是没时间学，但都是按钮界面，会Office的也好学，不用太长时间。最牛的是永远正版还免费。

微软套装里还有一个常用的是Visio，这也是好多人不愿意转变的原因，其实此类软件不是微软开创，在Mac下的OmniGraffle或origami比微软的强大很多，这是我用过很久的软件。linux下此类软件更多，只要搜索"linux visio"就会找到大量软件，我不做推荐，因为没用过。

2. 绘图软件：

主流绘图软件都有linux和Mac版本，包括常用的PS，甚至大部分3D绘图也都有。并且linux下有一个gimp与PS极其类似，这是我用过的，而且还是开源免费软件。
其实Adobe公司几乎所有软件在三大主流操作系统上都可以运行。这方面是不用担心有转换成本的。

3. 邮件接收：

其实这个是最不用担心的，因为各大操作系统都将此软件做为自带软件随操作系统一起发布。而且此类软件的功能单一，都没什么太大的区别，转换只是一个习惯性问题。

4. 压缩解压：

这个是要重点说一下的，大陆用户常用RAR或haozip来压缩解压，其实这里的haozip是背景有问题极度威胁安全性的软件。
Mac下可以用"eZip"替换，而没有压缩需求，平时只是解压的，可以考虑用"The Unarchiver"和"RAR Extractor Lite"，也是免费软件。Linux下可以用"PeaZip"替换。
其实Mac和Linux真正强大的压缩功能都在命令行里，上边推荐的都是当转过来时还不习惯命令行的软件。

5. 视频播放：

VLC是一款支持三大操作系统的播放器，在windows里习惯一下就可以直接转过来了。
MAC下视频播放软件好的软件"IINA"，是我目前用的较多的，感觉较好的。

6. 下载软件：

迅雷支持windows和Mac，短期可以不转。但长期来说还是建议转向aria2或BiglyBT，这种支持多协议的下载软件。linux下也有aria2。

7. 文本编辑：

三大操作系统都有简单文本编辑软件，跟随发布。如果有更高级的编辑需求，如：正则等功能，可以考虑UltraEdit、Sublime或Vim，都是支持三个系统，都可以先在Win里熟悉再转过来。Mac下还有Atom较流行。

8. 社交媒体：

Mac上QQ和微信都可以用，但linux版本曾经发过，但后来停止更新，所以，就算能用也需要较强的技术水平来安装。而且也没有网页版本。
其实战友完全可以在手机上用此类软件。战友也知道这类软件不安全，所以完全可以转到战友常用的勾通软件上来。这方面不用我说了，可能我还没有战友们知道的多。

9. 网页浏览：

这方面不用担心，chrome / firefox 都是支持三大操作系统，而且功能完全一样。

10.翻墙软件：

一直建议战友用V2ray，不要用VPN，所以这里只说V2ray的替换。
MAC：ClashX / V2rayX / Qv2ray / V2rayU
linux：Qv2ray / V2rayA / V2rayL / Mellow

11.文字输入：

鼠须管输入法是开源的安全输入法，支持三大操作系统，有拼音和五笔。但mac用户不用强制转，因为支持的输入法有好多。

三、系统差异：

linux因为是开源，所以对其它操作系统的兼容相对较好。而Mac因为是公司行为无法明显支持NTFS(微软硬盘格式)，可以安装"Tuxera for NTFS / Mounty for NTFS / paragon for NTFS来解决。
至于某些对黑客感兴趣的战友则必须向linux转，因为大部分软件都是先发或只发linux版本。而windows的权限制度根本无法满足黑客要求。
再说下Mac安装，好多人认为Mac只能升级，不能安装，其实不然。Mac安装一般只要一个16G的U盘就可以安装。制作此U盘最好是在操作系统更新时，新系统已经下载还未安装时，将此下载拷贝出来并用命令(网上大量教程)制作U盘，而不是去网上下载操作系统包，以防止操作系统被修改添加后门。
如果远离win操作系统，平时可多关注开源软件，因为有大量免费开源软件可用。而在Win中，此类软件基本都收费，而且价格不菲，Win中大量收费软件都是参考这些开源软件。
Mac的录音录屏无法同时，可以下载soundflower，用QuickTime实现。我发过教程。
因Mac内核是linux内核的修改版，所以，好多linux软件都有Mac版，或是可以在Mac上运行。

四、命令学习：

这是我的建议，因为这是linux操作系统的核心，只有学会用命令，转向后才不会去找大量的软件，因为好多windows用软件完成的工作，在linux中是用组合命令完成的。这些命令是原生的，不用另外下载。
如果有想提高的战友可以学习。这样可以在爆料革命中担当更多的工作。Mac与Linux的命令有细微差别，但不大，所以，用心学好一个，再转到另一个很容易。
其实命令的入门不难，一但入门就会改变对电脑程序的看法，对电脑操作也有一个新的认识。

如果有精力的战友可以学一下"正则表达式"，这是一种规则极其简单的用法，学习用时短，但正则体现了编程的所有精髓，能够快速提高工作速度。我曾经用正则快速(大概用20分钟，排版了几个文档合并的文档)排版几百页的Word文档。
