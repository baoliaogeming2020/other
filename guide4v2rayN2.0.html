v2rayN 是Windows平台上一款基于v2ray核心的简洁好用、功能强大的v2ray客户端，支持vmess / vless / shadowrocks / socks / trojan等多种协议，也支持服务器订阅。

本文将根据长期以来战友小白提出的问题撰写针对性较高的教程，希望战友能够将此教程向墙内传播，带动更多的墙内被长期洗脑的人民翻墙出来看看外面的真实世界。

本文作者长期在爆料革命战友DISCORD交流群里教墙内翻墙，时常发布节点文档，将此教程与节点文档一同传播基本可以保证墙内对外边世界有好奇心的人翻墙成功，并有可能成为我们的战友。

一、安装说明：

因V2RayN升级到4.29所以升级此教程，有战友按老教程操作了好几天也无法学会翻墙，故升级。
翻墙软件V2RayN是基于Win操作系统，但墙内大部分操作系统都是安装的Ghost版本，也就是那种免费版，包括:番茄花园、雨林木枫等，这些都是预装墙内杀毒软件和监控插件的操作系统，如果你正好用的是这种操作系统，在安装V2RayN时可能无法启动，或者报错。一般会弹窗提示"V2RayN已停止工作"，当遇到这种提示时，基本是由杀毒软件或监控插件引起。同时，此类操作系统还会预装一些垃圾软件，当你使用时会不断的安装其它垃圾软件，最终把你的电脑安全级别降到最低。
解决办法：
网上下载非Ghost版本操作系统，制作安装U盘，在网上找一套序列号。提醒：因为发了新版操作系统，微软现在已经不怎么用Win10挣钱，所以微软官网就可以下载原版IOS包，用这种包安装才是没被中共安装监控插件的操作系统。而网上有大量的Win10序列号公布出来都是可用的。
安装了干净的操作系统后，杀毒软件是一个问题，因为，好多墙内不习惯在网上裸奔(无防毒软件)，但一定不要安装360等墙内生产的杀毒软件，如果不想裸奔，最好找国外免费的，如：卡巴斯基、赛门铁克、诺顿，如果你是电脑发烧友，也可以找一些小众的，因为现在有好多此类软件。
如果，你感觉以上内容对你来说比较难操作，也可以选择虚拟机安装操作系统，然后在虚拟机里翻墙。虚拟机软件包括：vmware / virtualbox / Parallels / Hyper-V等。

二、软件下载：

v2rayN是开源软件，主要发布网站在github.com，而github是中共大部分程序员学习编程的主要网站，因此，长期以来中共都不太敢封锁此网站。如要下载v2rayN请直接访问release，选择最新版本"v2rayN-Core.zip"下载。此压缩包已经包涵最新版本"v2ray-core"，而"v2rayN.zip"是不包括"core"的版本，需要自行下载。而，无"core"是无法运行的。下载过程中一定要认真阅读此版本的提示信息，这些提示都是对安装有一定帮助的。
解释：
core：V2Ray的创史人只写了一个非界面版本的软件，也就是core，但对于小白来说很难学习和使用，而V2RayN，只是把这个Core界面化，而core的升级由创史人维护，所以，当下载的V2RayN没有core时，虽然能够启动，但无法实现翻墙。

三、软件安装：

1. V2rayN是非安装软件，解压到一个自己比较舒服的文件夹就可以。
2. 进入文件夹双击"v2rayN.exe"，V2rayN的图标(LOGO)出现在系统托盘中。
(WinOS的系统托盘就是整个屏幕最下边横条靠右的部分 ，其中第一个图标就是v2rayN。在这里大概介绍一下logo的颜色："红"代表节点失效无法连接，"蓝"代表软件正常，可能是连接成功，也可能是你关闭代理。)

四、简单翻墙：

1. 双击托盘LOGO -> 显示主界面。
2. 从其它软件复制"vmess://..."文本，可一次复制多个，如：我发的"vmessAll4KillCCP.html"文档，如果文档内包涵其它信息，无法一次性复制多个，可以将节点一个个复制到其它文本文档内，再一次性复制，点击"服务器 -> 从剪贴板批量导入"。最好多加几个节点，因为不是所有节点都能穿透你的网络。
也可以通过添加订阅链接，一次性获取多个节点，参阅后边的"订阅"。
3. 服务器列表 -> 点击某个节点 -> 按"Enter"(回车)键。
4. 右键点击系统托盘中V2rayN的LOGO -> 系统代理 -> 自动配置代理。
5. 如果无法翻墙，可能是节点问题，重复步骤3-4，切换其它节点试试。也可以修改"系统代理"和"路由"里的其它项试试。
路由三项说明：
绕过大陆：需要设置，也就是白名单，一般写入需要翻墙的网址。
黑名单：需要设置，只有黑名单(一般写入墙内网址)中的不通过翻墙软件代理。
全局：你上网的所有链接都会通过翻墙软件代理，就算是连接墙内网站。

五、界面简介：

V2rayN启动后直接进入系统托盘，在托盘内双击才会出现主窗口，右键会出现托盘菜单。

1. 工具栏：

服务器：

添加[****]服务器：[****]中可以是vmess / vless / shadowrocks / socks / trojan，其中，vmess / vless / trojan是比较安全的，有加密的协议，shadowrocks就是觉的SS协议，此协议已经可以被GFW(中共的墙)发现，所以现在用的较少，而socks是一种很古老的代理协议，虽然也有加密，但安全性很低。当节点添加后可以在"服务器列表"的第二列中看到协议的类型，尽量选择三类安全的协议，保证翻墙的安全。
此菜单会弹出窗口，将以手动方式添加服务器节点，用此菜单大部分为自建节点的战友，因为自建节点会生成节点信息的每一项，而不是vmess文本。因此，需要自己将每一项信息手动一步一步填写。
从剪贴板批量导入：先从其它软件复制vmess文本信息，要求每行一条。然后点击此菜单项，会在窗口中部"服务器列表"中增加节点。如果剪贴板复制内容无效，不会提示信息。
扫描屏幕上的二维码：先打开包括vmess二维码信息的图片或者网页，然后点击此菜单项，本软件会关闭主窗口，以保证二维码出现在屏幕，扫描成功会在窗口中部"服务器列表"中增加节点。如果二维码无效，会提示"扫描完成，未发现有效二维码"。
测试服务器延迟Ping：以TCMP网络协议测试服务器，并在服务器列表的每个节点的"测试结果"列表项中显示本节点测试时间。但本测试只表明服务器开机，并不代表一定能够连接代理成功，因为v2ray走TCP网络协议。
测试服务器延迟Tcping：以TCP网络协议测试服务器，并在服务器列表的每个节点的"测试结果"列表项中显示本节点测试时间。本测试代表服务器代理连接时间延迟。
测试服务器直连延迟：直接连接"google.om"并报告延迟时间，此测试更准备表明本服务器的连接性。但是，某些服务器IP访问某些网站时速度极慢，所以，如果你要访问的网站代理速度极慢，请多换几个节点试试。
设为活动服务器：当"服务器列表"中不只有一个节点时，选择其中一个节点，点击此菜单项，会将此节点设置为活动服务器，也就是所有代理通过此节点连接上网。相当于按"Enter"(回车)键。
菜单其它项："上移、下移"等，主要是移动"服务器列表"中节点的排列顺序。"导出、批量导出"等，主要是将选择的节点或订阅导出到剪贴板，此类菜单项主要是为了向其它人分享节点信息。

快捷键：大部分常用菜单项后会有快捷键，不用专门记忆，用多了就会记住，比如最常用的"设为活动服务器"对应"enter"，其实就是回车键。

订阅：

因v2ray是最流行的翻墙软件，同时网上有非常多的节点订阅链接，此类链接以https链接形式发放，此链接会下载一个包涵加密vmess节点的文本。当把此类订阅链接填写到软件时，本软件会解析出链接中的所有vmess节点并添加到"服务器列表"中。因中共防火墙的封锁，此类订阅会不定期更新节点信息，因此，个人电脑也要时常更新，以保证节点代理翻墙的有效性。
订阅设置：弹出窗口，每点击一次"添加"按钮会增加一组"订阅详情"，"备注"随意填写，"地址"填写"https"订阅链接。"移除"按钮为删除本条订阅。"启用"单选框选择，表明在更新订阅时更新本条订阅链接。最后点击"确认"按钮关闭窗口。
更新订阅：更新"订阅设置"窗口中启用的订阅链接。更新订阅也需要翻墙，也就是第一次用此软件翻墙，无法使用订阅，除非你更新订阅时有其它翻墙方式辅助。当你用失效节点更新订阅时也不会成功。

分享：

此项无下级菜单，点击此项后，会在窗口中部，"服务器列表"右侧出现选择节点的二维码与vmess文本信息，当向战友分享节点时可直接截屏二维码信息(适用于对方是手机)，或者，复制vmess文本(适用于对方是电脑)，并向战友发送。也可以用电脑测试节点，然后手机扫码添加到shadowrocket中(适用于自己有电脑和手机)。

设置：

参数设置：此项会弹出窗口，窗口内包涵多个标签，但对于普通战友只讲一项最有用的。"Core:基础设置->本地监听端口"里的数字。当某些软件需要手动添加代理时，一般的格式是"127.0.0.1:10808"，这里的10808就是这里的数字，在这里可以修改，但一般最好别改，容易跟其它软件冲突。
路由设置：这里就是上边提到的路由三项的设置，双击一项后会弹出窗口进行设置，不多讲，因为太难讲懂了，如果有兴趣可以自己搜索，网上有大量做好的黑白名单可以直接调用，很方便。

2. 服务器列表：

此项只介绍几点有用的，其它不介绍。
双击一个节点，弹出"编辑或添加[vmess]服务器"窗口，此窗口中"底层传输安全"项最好为"tls"，才能保证翻墙的安全性。此选项不是任意选择的，只有服务器节点设置了此项功能，选择"tls"选项才能保证连接。如果是通过vmess添加节点最好要查看此项是否为"tls"，如果不是则最好少用。
加密方式：此项选择最好不是"none"，此项是对代理传输内容的加密以增加GFW(长城防火墙)封锁IP的难度。

3. 信息：

信息栏会显示所有连接信息，当所有设置都正常但还是无法翻墙时可以查看此栏信息，关键查看最后几条信息，将信息内容复制并在谷歌中搜索，一般都能找到解决办法。

4. 状态栏：

SOCKS5: 127.0.0.1:10808：当某些软件无法自动走系统代理时，要通过手动设置，如果走socks5代理就要在软件内填写此内容。
http: 127.0.0.1:10809：当某些软件无法自动走系统代理时，要通过手动设置，如果走http代理就要在软件内填写此内容。
PAC: ON/OFF：表明当前是全局代理(OFF)还是非全局代理(ON)。
网速显示未启用：表明当前代理不可用。

六、常见问题：

1. 如何判断订阅链接的安全性：用浏览器打开链接，看到的文本以"dm"开头就是"vmess"链接，安全性不错，如果以"c3"开头就是"ss"或"ssr"链接，安全性不好。
2. 自建节点无法连接最可能问题：最可能的是服务器与本机时间不协同，因v2ray要求时间差在2min中内，所以，如果你本机时间是网络同步，就要改服务器节点的时间。也有部分战友本机时间是手动设置，甚至年月都与当前时间不匹配。
3. 什么是翻墙的最后一道保障：答案是telegram或者i2p。因为telegram有一种mtproto代理，这种免费的独特代理数量及其庞大，好多telegram群为了推广都会不断推出此类代理，同时有大量发布vmess节点的群。i2p，类似tor，但不用获取节点。也有些人说最后一道保障是其它，我解释一下，比如：tor，国内只有联通速度还可以，但难以获取节点。

七、节点格式：

v2ray翻墙节点有6种格式，这些格式一方面方便了节点的传播和添加，但也让大部分战友迷惑。获取错误格式通常会在添加节点时提示错误。
1. 通用链接格式："vmess://..."，此种格式全部内容都是base64加密，无明文，也就是看不懂的乱码。这种格式适用大部分v2ray软件。
2. kitsunebi格式："vmess://...?network=tcp&aid=0&tls=0&allowInsecure=1&mux=0&..."，此格式前半部分是base64加密，后边是明文参数，也就是"?&a=b"。此类格式只适用kitsunebi。
3. BifrostV格式："bfv://45.63.34.187:50750/shadowsocks/1?rtype=lanchina&dns=8.8.8.8&method=..."，此格式全是明文参数，此类格式只适用BifrostV。
4. v2ray订阅链接：订阅链接内容是对多个vmess再次base64加密的文本以http或https发布，一般都是"https://..."。此种链接里的内容也可能会出现以上三种vmess的格式，大部分都是第一种，也有把SSR加入其中的。
5. clash订阅链接：clash专用的订阅链接内容不详细说了。这种格式太大了。此格式只适用clash。
6. 二维码：1 - 3都可以生成二维码，但不同格式生成的码只适用于原格式软件扫码。
