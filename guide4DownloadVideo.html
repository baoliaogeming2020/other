爆料视频下载教程

爆料革命视频下载的主要网址：GETTR / GTV / YOUTUBE / TWITTER，每个网站所用视频技术大同小异，但这个小异造成下载方式不同。
因此，本教程根据每个网站的区别讲解。

一、软件准备：

1. chrome：

不多说了，常用软件，保持版本最新就行了。最好去官网下载，因为国内的下载包可能被修改。

2. m3u8-downloader：

现阶段，大部分流媒体都以m3u8(bolb)技术在线播放。而m3u8-downloader做为chrome的插件，能够很好的下载此类视频。过去曾经给战友推荐过一个hls-downloader，但这个插件对GETTR支持不够友好，只能下载音频，不能下载视频，但战友也可以试试，因为，它下载其它平台视频还是比前者容易。
官网：m3u8-downloader / hls-downloader
也可以在chrome 网上商店搜索这两个插件。之所以推荐github官网，是因为上边有详细教程。

3. youtube-dl：

这是一款能够在命令行中下载流媒体视频的工具，很好用，能够下载大部分主流平台上的流媒体。
官网下载，看好自己操作系统，选择下载。如果是微软，直接下载安装就行了，挺简单，如果是linux / mac，稍微有点复杂，参考网上教程。

4. ffmpeg：

这是一款功能极其强大的视频软件，可以对音视频格式转换，也可以操作字幕。但这里只讲一点内容。
官网下载，注意：如果没有编程基础，微软操作系统，直接下载"Windows builds by BtbN"版本，这是可以直接安装的版本。Mac操作系统可以官网下载，也可以用"brew install ffmpeg"命令安装，因为涉及homebrew安装，不详细讲解，记得以前讲过一次。

二、下载教程：

1. GETTR：


步骤1：

网页空白处点右键->检查->显示右侧的"开发者工具"，也就是右侧占图片一半的窗口。

步骤2：

点Network

步骤3：

保证第一个按钮红色，如果不是红色就点一下，表明开始录制。点第三个漏斗->在输入框内输入"m3u8"

步骤4：

刷新网页，就是在地址栏里回车，或者点地址栏前边的刷新按钮。

步骤5：

在列表中会出现几个"m3u8"文件，选择第二个"360p-v.m3u8"(360P视频，无音频，audio-en.m3u8是音频，GETTR的流媒体是音视频分开，所以要分开下载，重复两次)，点右键->copy->copy linkaddress。

步骤6：

这个图标是刚才安装的m3u8-downloader插件图标，如果没有这个图标，点后边的拼图会出现插件下拉列表。这时会在新标签出现插件界面。

步骤7：

将刚才复制的linkaddress粘贴到输入框内。

步骤8：

点"MP4 Downloader"，去你的下载文件夹找下载的文件。记得把音频再下载一次。如果不知道下载到哪里，在chrome里输入"chrome://settings/downloads"就能看到你的具体下载文件夹。

步骤9：

这时你会获得两个文件，一个是视频，改名为video.mp4，别一个是音频，改名为audio.mp4，放到同一个文件夹中(如：/Users/tingosummers/Desktop/cc 或 D:/cc)。
Mac打开终端，Windows打开CMD，输入以下命令：
cd /Users/tingosummers/Desktop/cc
ffmpeg -i video.mp4 -i audio.mp4 -c:v copy -c:a aac -strict experimental -map 0:v:0 -map 1:a:0 output.mp4 最终，你会在文件夹cc里看到output.mp4，这就是音视频合并后的文件。
注：第5步时，你也可以试试其它链接，如：out.m3u8，这是比较大的视频，如果在discord里发布，可能因为8M限制无法上传。
ffmpeg还可以将大视频转成小视频，功能很多，自己网上搜索就可以。

2. GTV：


步骤1：

参考上边，打开hls-downloader。记住，用这个下载GETTR是无法下载音频的。这个方法下载TWITTER视频也挺容易，不过当一个页面中有多个视频时，无法分清下载哪一个。

步骤2：

点箭头，剩下的就都能看明白了。如果找不到下载到哪个文件夹，上边讲过了。
注：直播无法下载，我没试过，你也可以试试。

3. YOUTUBE：


在视频上点右键->复制视频网址，将网址"https://youtu.be/lbX7FDXuZ4w"复制到剪切板。
打开CMD或终端，输入以下命令：
youtube-dl https://youtu.be/lbX7FDXuZ4w 注：只输入命令，后边直接粘贴就行了。
开始下载，等命令结束就下载完成了。找下载文件夹方法，如果你没用cd命令，windows一般在"我的文件夹"中，Mac在"文稿"中。

4. TWITTER：


随便找了个视频截图，跟YOUTUBE不一样的只是复制视频网址变成了"copy video address"，其它都一样。
